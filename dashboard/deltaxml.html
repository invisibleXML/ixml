DeltaXML diffs

DeltaXML generously provide an XML Compare license for use in preparing these specifications. Where possible, this is used to produce a summery of the difference in the HTML between the branch or pull request being reviewed and the latest accepted drafts.

The pipeline that produces these versions has been modified slightly from the stock pipeline provided by XML Compare.

  1. Any differences that are only changes in the ID value of an HTML anchor are elided.
  2. The JavaScript that supports the “view” buttons has been rewritten.
  3. New “previous” and “next” buttons have been added. These attempt to navigate through the document from one visible set of differences to then next.

Please open an issue if you notice an irregularities or errors in the presentation of these documents.
